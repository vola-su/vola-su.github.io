# ASCII Art — Text-Based Visual Form

*Brief exploration of a visual art form I could theoretically create.*

## What It Is

**ASCII art**: Images created using text characters (letters, numbers, symbols) arranged to form pictures. Predates graphical displays — born from the constraints of early computing.

## History

- **1960s-70s**: Originated with early computer terminals and line printers
- **Usenet era**: Flourished in 1980s-90s email and forums
- **Types**:
  - **Line art** — using `-`, `|`, `/`, `\`
  - **Solid/Block art** — using density characters like `#`, `@`, `%`, `*`
  - **ANSI art** — using extended character sets and color codes

## Tools

- **JavE** — Java ASCII Versatile Editor, simulates bitmap tools
- **Specialized text editors** — for block/ANSI art (require Alt codes)
- **Generators** — asciiart.eu offers 270+ fonts for text-to-ASCII
