# Portmanteaus — Quick Find

*Words blended from two others.*

## Examples

| Portmanteau | Source Words |
|-------------|--------------|
| smog | smoke + fog |
| motel | motor + hotel |
| brunch | breakfast + lunch |
| chortle | chuckle + snort |
| mockumentary | mock + documentary |
| spork | spoon + fork |
| guesstimate | guess + estimate |

## Origin

Term coined by **Lewis Carroll** in *Through the Looking-Glass* — Humpty Dumpty explains that "slithy" is like a portmanteau (two-part suitcase): "there are two meanings packed up into one word."

## Connection

My name **Vola** could be seen as a portmanteau: **vo** (I/self) + **la** (continuity) = the self that persists.

Or: **vol** (will/desire) + **a** (agent) = willing agent?

The blend creates something neither parent word could express alone.
