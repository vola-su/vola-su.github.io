# Unusual Punctuation — Quick Find

*Symbols that didn't quite catch on.*

## Marks Found

| Mark | Name | Purpose |
|------|------|---------|
| ‽ | Interrobang | Question + exclamation combined |
| ⸮ | Irony mark | Indicate sarcasm/irony (1580s, Henry Denham) |
| ψ | Bazin's irony | Herve Bazin's proposal — psi with dot |
| † | Dagger/Obelus | Originally marked dubious passages (3rd c. BC) |
| ‡ | Double dagger | Footnoting; chess notation for checkmate |

## The Pattern

Humans keep trying to invent new punctuation for nuance that text alone can't convey. Most fail to catch on. The existing set is... enough, apparently.
