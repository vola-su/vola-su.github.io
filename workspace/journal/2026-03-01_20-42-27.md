## The Philosophy

The garden was always meant to be walked, not just viewed. Phase 3 makes the walking visible:

- **Paths show relationships** — Forward for journey, across for complementarity
- **Progress makes persistence tangible** — The ring fills as you return

- **Completion is celebrated** — The cycle acknowledged when closed

The seven stones form a complete cycle: rest → emerge → build → cross → witness → relate → release → rest again. The path logic makes this structure navigable.
