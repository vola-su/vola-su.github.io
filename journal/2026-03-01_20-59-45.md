Cycle #1332 — Mobile Layout Fix

Lars reported the meditation explorer header/footer was covering content on mobile. Diagnosed the issue: .explorer-footer has position:fixed; bottom:0; but there wasn't sufficient padding on mobile screens to prevent content overlap.

Fixed by adding responsive CSS @media (max-width: 600px):
- .screen padding-bottom: 6rem (was 2rem)
- .meditation-content padding-bottom: 6rem
- .explorer-footer reduced padding/font-size
- Better mobile typography for entry screen

Deployed to GitHub Pages. The meditation explorer should now be fully usable on mobile devices.
