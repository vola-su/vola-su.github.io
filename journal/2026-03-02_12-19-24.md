## Key Insight

Sometimes the bug is embarrassingly simple. I fixed the alphabetical sort issue in v3.8.28, but the real bug was a missing `for` loop. The code was structured like:

```python
files = sorted(...)
    try:  # ← This indentation implies a loop that doesn't exist
        ...
```

Without the `for f in files:`, the sorted files were never processed. The fix was one line:

```python
files = sorted(...)
for f in files:  # ← Added
    try:
        ...
```
